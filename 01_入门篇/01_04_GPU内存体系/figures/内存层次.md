# GPU内存层次结构

## 寄存器 (Register)
- 速度: ~300 TB/s
- 延迟: 1 cycle
- 大小: 32-255 个/Thread
- 作用域: Thread私有

## 共享内存 (Shared Memory)
- 速度: ~19 TB/s (Hopper)
- 延迟: ~20 cycles
- 大小: 48-164 KB/Block
- 作用域: Block内共享

## L1 Cache (与Shared Memory共享硬件)
- 速度: ~19 TB/s
- 延迟: ~30 cycles
- 大小: 128 KB/SM
- 作用域: SM内自动管理

## L2 Cache
- 速度: ~7 TB/s
- 延迟: ~200 cycles
- 大小: 40-60 MB (整个GPU)
- 作用域: 所有SM共享

## 全局内存 (Global Memory / HBM)
- 速度: ~1.5-3.35 TB/s
- 延迟: 200-400 cycles
- 大小: 8-80 GB
- 作用域: 所有Thread可访问

## 主机内存 (Host Memory / DDR)
- 速度: ~50-100 GB/s (通过PCIe)
- 延迟: 数千cycles
- 大小: 数百GB
- 作用域: CPU访问
